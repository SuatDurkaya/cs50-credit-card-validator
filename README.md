# Credit Card Validator (CS50)

A C program that validates credit card numbers using Luhn’s Algorithm
and identifies the card type (AMEX, VISA, MASTERCARD).

## Concepts Used
- Loops
- Functions
- Modulus arithmetic
- Luhn Algorithm

## Example
Card Number: 378282246310005  
Output: AMEX
